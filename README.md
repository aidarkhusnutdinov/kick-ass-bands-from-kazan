# Интересные казанские группы

Статичная веб-страница — путеводитель по независимой музыкальной сцене Казани.
23 группы: фото, описание, ссылка и трек для быстрого прослушивания (плеер YouTube).

**Живая страница:** https://aidarkhusnutdinov.github.io/kick-ass-bands-from-kazan/

## Структура
- `index.html` — вёрстка и стили
- `data.js` — данные о группах (название, фото, описание, ссылка, видео)
- `images/` — фотографии групп

## Источники
Описания и фото собраны из телеграм-канала фестиваля «ВСЕ» [@fest_vse](https://t.me/fest_vse),
материала [«Инде»](https://inde.io/article/96362-novaya-muzyka-iz-tatarstana-80-ispolniteley-kotoryh-stoit-slushat-v-2025-godu/)
и канала [@kazanalization](https://t.me/kazanalization). Фото «Триггерной точки» — [@myfriendaidar2](https://t.me/myfriendaidar2/251).

Никакой сборки не требуется — просто откройте `index.html`.
